# 🚀 Ultimate Media Organizer v5.0 (Final)

Профессиональный инструмент на Python для наведения порядка в медиа-архивах любого объема (протестировано на 4 ТБ).

## ✨ Что умеет этот скрипт:
- **Умная сортировка:** Раскладывает файлы по папкам `Год / Месяц` (напр. `2024 / 05-May`).
- **Двойная проверка дубликатов:** Сравнивает файлы по размеру и MD5-хешу (содержимому). Разные фото с одинаковым размером НЕ удаляются.
- **Поддержка форматов:** - Фото: JPG, PNG, HEIC, BMP, WEBP, GIF, TIFF, PSD.
  - Видео: MP4, MOV, AVI, MKV, WMV, MPG, 3GP.
- **Очистка мусора:** Удаляет `.DS_Store`, `Thumbs.db`, ярлыки `.lnk` и пустые папки.
- **Безопасность:** Работает через перемещение (`shutil.move`), ничего не теряет при сбоях.

## 🛠 Как пользоваться:
1. Установите Python 3 и библиотеку Pillow: `pip install Pillow`
2. Запустите скрипт: `python3 ultimate_media_sorter.py`
3. Укажите путь к "Хаосу" и путь к "Золотому архиву".
4. Если Mac уходит в сон, используйте команду `caffeinate -u -t 36000` во второй вкладке терминала.
